# Description

  This script for viseven devs.

## Engines

  [NodeJS](https://nodejs.org/en/) >=11.13.0

## Installing

  * git clone git@github.com:ssm1le/viseven-cli.git
  * cd viseven-cli
  * npm run link

## Usage

    vis -h, --help
    
  Remove duplicates slides:
  
    vis img [path to move images]
